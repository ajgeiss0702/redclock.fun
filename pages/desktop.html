Desktop app

The redclock.fun desktop app is here and ready!

Red Clock Desktop is a program that will show you the time no matter what you have open! When you first open the installer, it will tell you it is unsafe. This is because i dont want to pay $80+ per year to get that message to go away. Click more info then Run Anyway.

Once you have it installed, it will put a shortcut on your desktop and open it. Once it is open, you will be able to move it around while it has the gray background. Once you click away from it, the background will go completely clear and you can do whatever you were going to do. If you want to move it around, you can move it while the background is gray. To go into this state again, just click the countdown.

While you are in the moving state, you can make it bigger or smaller by pressing + and -. Press 0 to make it go back to the default size.

To close it, just press escape while the background is gray.

Join the discord server to tell me about any bugs you encounter!
